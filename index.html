HTTP Status Codes - Cheat Sheet

📥 Network Packets Legend

Guide to incoming data packets on the network stream:

  • 🟢 200 OK (Standard): The core code packet. Increases your score by 10 points, expands your file buffer (length), and feeds the stream.
  • ⚡ 404 Not Found (Anomaly): The glitched file. Multiplies your current score, but triggers visual noise, screen distortion, and inverts controls for 5 seconds. High risk, high reward!
  • 🌀 301 Redirect (Wormhole): A redirection packet. Instantly moves the snake's head to a random portal elsewhere in the grid. Keeps length unchanged.
  • 🚫 403 Forbidden (Firewall): Blocked node. Spawns wall blocks dynamically. Crashing into one triggers an immediate termination (Game Over).
  • 🔵 500 Server Error (Crash): Fatal server anomaly. Triggers a full system crash overlay (Blue Screen of Death) for 2 seconds. The server automatically recovers, shrinking the snake's length by 3 segments and resetting current speed modifiers.
