HTTP Status Codes - Cheat Sheet

📥 Network Packets Legend

  • 🟢 200 OK: Core data packet. Grows tail and increases speed.
  • ⚡ 404 Not Found: Anomaly. Inverts controls and shakes screen for 5 seconds, but boosts score.
  • 🌀 301 Redirect: Redirection. Teleports snake head to a random cell.
  • 🔵 500 Server Error: Severe crash. Triggers a Blue Screen of Death, shrinking tail by 3 segments and resetting speed.
  • 🚫 403 Forbidden (Firewall): Impassable red hazard nodes. Colliding triggers Game Over.

🛡️ HTTP Power-up Packets

  • 💛 HTTPS (SSL Shield): Grants a protective aura that absorbs 1 collision (self or firewall).
  • 💙 gzip (Decompress): Compresses your snake buffer, cutting tail length by 3 segments.
  • 🩵 Cache-Control (Slow-Mo): Slows down the speed of game ticks temporarily.

👾 Malware Bug (Hazard)

  • 🔴 Malware Bug: A wandering red block that crawls on Staging and Production levels. Touching it crashes the server.
